# 📧 Spam Email Classifier using Machine Learning

## 🚀 Project Overview
This project is a **Spam Email Detection System** that classifies emails into:
- ❌ Spam  
- ✅ Ham (Not Spam)  

It uses **Machine Learning (Naive Bayes)** to detect spam messages based on text content.

---

## 🧠 Features
- Detects spam emails
- Real-time prediction
- Fast and lightweight model
- Easy to understand implementation

---

## 🛠️ Technologies Used
- Python 🐍
- Pandas
- Scikit-learn
- CountVectorizer
- Naive Bayes Algorithm
